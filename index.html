Conseptos Esenciales Del Desarrollo Web


1. Que es HTML

Hyper Text Markup Language o Lenguaje de marcas Hipertexto. Es el componente mas basico de la web. Define el significado y la estructura del contenido.


2. Etiquetas

Las etiquetas de HTML van a ayudarnos a brindar una estructura y semantica al contenido de nuestra website y cada una tiene caracteristicas y usos diferentes aunque visualmente den un resltado similar.

Muchas etiquetas son facil de diferencias como los headings o las imagenes y otras diferentes como header, fiiter, section o div, aqui es donde la semantica sera la respuesta y si sabemos un poco de ingles tambien podremos darnos unas pequena idea de su uso pero en este curso vamos a usar las mas importantes y que seran parte de tu dia a dia como software Developer.


3. DOOM

El Documento Object Model es una estructura de arbol que representara todos nuestros proyectos web como si un arbol genealogico fuera: padres, hijos, hermanos con niveles infinitos. cuando entendemos esta anidacion podemos identificar dependencias, herencias en css y que tan complejo es nuestro proyecto.



4. Semantica

La semantica le brinda sentido a cada elemento existen en sitio web, algunos elementos sera meramente decorativos y no deben significar nada pero otros seran titulos y deberan estar en diferente jerarquias o alguna secciones seran mas relevantes que otras, esto ayudara a los motores de busqueda como Google o Duck Duck Go a diferenciar tu contenido, categorizar y sera la herramientas mas valisoa para estar en los primeros resultados de busqueda SEo. Tambien ayudara a la accesibilidad de tu sitio web, para que personas con hablidades diferentes pueden entender cada contenido.


5. Atributos

Los atributos le dan caracteristicas extra a las etiquetas para complementar informacion y son pre programadas por el navegador, algunos atributos funcionan en todas la etiquetas de la misma forma como 'class' pero otros atributos funcionan en solo alguna etiquetas como src

6. CSS

Hojas de estilo en Cascada (del ingles Cascading Style Sheets) o CSS es el lenguaje de estilo utilizado para desecribir la presentacion de documentos HTML
